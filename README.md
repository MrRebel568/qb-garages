# qb-garages


This is a qb-garages script that uses the radialmenu to retrieve and park vehicles.
Almost everything is fully customizable to the last bit!


## Dependencies
 - [qb-radialmenu](https://github.com/qbcore-framework/qb-radialmenu)
 - [qb-core](https://github.com/qbcore-framework/qb-core)
 - [PolyZone](https://github.com/mkafrin/PolyZone)

## Installation

- Drag and Drop replace for qb-garages.
- Delete qb-garages.
- Drag the downloaded qb-garages folder into the [qb] folder.

## Features

* Public Garages
* House Garages
* Gang Garages
* Job Garages (Supports only owned cars for now...)
* Depot Garages
* Custom DrawText
* Water Garages
* Aircraft Garages
